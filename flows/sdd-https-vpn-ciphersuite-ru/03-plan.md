## План работ

### Фаза 1: Низкоуровневые примитивы (ГОСТ)
- [x] Реализация Кузнечик (Grasshopper) и Магма (Magma)
- [x] Реализация Стрибог (Streebog)
- [x] Реализация ЭЦП ГОСТ Р 34.10-2012
- [x] Поддержка кривых tc26

### Фаза 2: Кастомный TLS уровень
- [x] Определение Cipher Suites ГОСТ
- [x] Реализация VKO обмена ключами
- [x] Реализация Record Layer для ГОСТ наборов
- [x] Логика Handshake для ГОСТ

### Фаза 3: Интеграция провайдера
- [ ] Создание `crypto/ru/provider.go`
- [ ] Регистрация провайдера в `crypto.Registry`
- [ ] Настройка TLS конфигурации для работы с кастомным стеком

### Фаза 4: Тестирование
- [x] Unit-тесты для примитивов (Kuznyechik, Magma, Streebog)
- [ ] Сквозное тестирование TLS соединения
