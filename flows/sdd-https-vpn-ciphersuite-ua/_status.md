## Контекстні нотатки

Ключові рішення та контекст для продовження роботи:

- Базується на README_UA_CRYPTOGRAPHY.md - постквантова криптографія за стандартом ДСТУ-ПК 2026
- Основні алгоритми:
  - **Мальва (Malva)** - KEM на базі Module-LWE (аналог ML-KEM/Kyber-1024)
  - **Сокіл (Sokil)** - цифровий підпис на базі SIS (аналог ML-DSA/Dilithium-5)
  - **Калина-ПК (Kalyna-PQ)** - блочний шифр 512-біт (модифікація ДСТУ 7624:2014)
  - **Купина-ПК (Kupyna-PQ)** - хеш-функція 512-біт (ДСТУ 7564:2014)
- Гібридний режим TLS 1.3: X25519/ДСТУ 4145 + Malva-1024
- Вся документація, код та спеки пишуться українською мовою

**Примітка для production:**
- Постквантові алгоритми (Мальва, Сокіл) мають базову структуру
- Для повної функціональності потрібна реалізація NTT
- Тестові вектори за ДСТУ потребують уточнення
